Books API — Web UI

A small single-file frontend to exercise the in-memory Books REST API running at http://localhost:3000.

Base URL

Make sure your API server (books-api-server.js) is running on http://localhost:3000. CORS may be required if you open this page from a file:// URL — run the server with CORS enabled or serve this page via a local static server.

    Quick update / delete

    Use the controls next to each book in the list to update its title/author or delete it.

    Example API calls (curl)

    # List
curl http://git:3000/books

# Add
curl -X POST http://localhost:3000/books -H "Content-Type: application/json" -d '{"title":"The Hobbit","author":"J.R.R. Tolkien"}'

# Update
curl -X PUT http://localhost:3000/books/1 -H "Content-Type: application/json" -d '{"title":"New title"}'

# Delete
curl -X DELETE http://localhost:3000/books/1
